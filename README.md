Wayfa

Ejecutar en Local
IMPORTANTE: Tener instalado node en su ultima version. 

1. Instalar dependencias

```bash
npm install
```

2. Ejecutar el proyecto

```bash
npm run dev
```

La aplicación estará disponible en `http://localhost:3000`
